# NetProbe

## 📌 Description

NetProbe is a simple Bash-based network scanner that finds active devices on a local network and scans them for open ports.

## ⚙️ Features

* Detects active hosts using ping
* Fast scanning with parallel execution
* Scans ports using Nmap
* Saves results automatically

## 🚀 Usage

```bash
chmod +x net_scanner.sh
./net_scanner.sh
```

## 📂 Output

* `output.txt` → list of active devices
* `result.txt` → port scan results

## 🛠️ Requirements

* Linux / Unix system
* Nmap installed

## ⚠️ Note

Use only on networks you own or have permission to scan.
